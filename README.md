# SEÇÃO 3 - Curso Web Design Fundamentos/Udemy

## Listas aninhadas 
> Uma lista dentro da outra

```html
<ul>
  <li></li>
   <li>
      <ul> 
         <li></li>
         <li></li>
      </ul>
   </li>
   <li></li>
</div>
```

## Listas de definição
> Um dicionário, por exemplo, onde temos um termo e uma definição.

```html
<dl>
  <dt></dt>
  <dd></dd>
  <dt></dt>
  <dd></dd>
</dl>
```
